## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:
* Instalar a versão mais recente de NodeJs (React, npm)
* Foi desenvolvido no Visual Studio Code


## ☕ Usando Clone Netflix

Para usar Clone Netflix, siga estas etapas:
* No caso do vscode, faça o clone do projeto
* No terminal: npm start

## 📫 Contribuindo para Clone Netflix
Para contribuir com Clone Netflix, siga estas etapas:

1. Bifurque este repositório.
2. Crie um branch: `git checkout -b <nome_branch>`.
3. Faça suas alterações e confirme-as: `git commit -m '<mensagem_commit>'`
4. Envie para o branch original: `git push origin CloneNetflix / <local>`
5. Crie a solicitação de pull.
